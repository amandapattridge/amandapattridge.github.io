Nodal error 'Fatal: role 'postgres' does not exist'

12 May 2016

This error occurs because Nodal looks for a postgreSQL user named 'postgres'. The postgres.app automatically sets your username according to your account name, so you need to check if you have a user called postgres with database creation authorization.

In psql, to view users:

yourname=> \du

If you do not see a user with role name postgres and an attribute of Superuser, you'll need to create that user.

CREATE USER postgres SUPERUSER;

That's it!
